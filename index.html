How It Works!

Step 1: Enter the website homepage URL.
Step 2: Select any one option from the "Search By".
  • By Categories - Select any of the category from the list. By default the list is of 25 articles.
  • By ID's - Enter the post ID of the particular article.
Step 3: Press the "Get Lists" or "Get Article" button as per the above selected option.

Note: This tools only works for the WordPress websites which has REST API enabled.
